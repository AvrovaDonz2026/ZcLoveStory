# 《社长恋爱物语》v1 完整台本

## 写作边界

- 玩家视角：记录者。
- 文风：悬疑克制，恋爱物语外壳，资料整理内核。
- 互动：资料路径选择只改变阅读顺序；完整主线按六章顺序连读。
- 公开包不放原始截图、PDF、聊天记录，只放匿名化、改绘、摘录和时间线重排后的资料卡描述。
- 不使用直接定性词；统一用“材料显示”“截图呈现”“聊天中出现”“旁观者视角”限定叙述边界。

## 主线顺序

1. `start.txt` 进入资料夹 hub，选择“完整主线”。
2. `ch01_keywords.txt` 关键词索引。
3. `ch02_evaluation.txt` 具体评价如何从事情滑向人。
4. `ch03_shared_story.txt` 争执之后仍被放回共同叙事。
5. `ch04_apr9.txt` 4月9日的分开复盘、旁观者视角和同日互动。
6. `ch05_apr19.txt` 4月19日的自责、道歉、安抚与日常回落。
7. `ch06_after.txt` 分开之后的强烈情绪、线下安排、组织事务和后续牵连。

## 资料入口

- `cast.txt`：角色职责与半匿名边界。
- `timeline.txt`：章节索引版时间线。
- `evidence_room.txt`：资料卡与证据柜。

## timeline.txt：时间线：章节索引版

```webgal
; 时间线：章节索引版

:时间线页。;
:这不是完整聊天记录，也不是替任何人下判词。;
:它把材料显示、截图呈现、聊天中出现、旁观者视角这四类线索，整理成可玩的章节入口。;

记录者:开始前先确认边界。;
记录者:这里不出现现实身份信息、机构名、社群标识和可回溯联络标识，也不引用截图路径。;
记录者:每一段只说明“材料里可以看到什么”，不把缺失的空白补成确定事实。;

旁观者:那玩家要怎么读？;
记录者:像读一张时间线。先看反复出现的词，再看具体事件，再把它放回共同叙事。;
记录者:最后看分开之后，话题如何从两个人的关系，继续扩散到组织事务和旁人评价。;

记录者:第一章，关键词。;
记录者:聊天中出现过“去死”“想死”“痛苦”这类高压表达。;
记录者:截图也呈现过“加油”“没关系”“我会陪着”一类安抚和鼓励。;
同学A:所以不是单纯挑难听的话出来？;
记录者:不是。关键词章节必须同时放入负面表达和安抚表达。;
记录者:玩家要看的，是高压表达出现时，谁在解释，谁在缓和，谁在承接后续情绪。;

记录者:第二章，具体评价事件。;
记录者:材料显示，某些评价从事情本身，滑向对整个人的否定。;
记录者:截图呈现的重点不是评价准不准，而是评价如何被说出、如何被接住。;
旁观者:如果只是一次争执呢？;
记录者:所以这里不单独停在一句话。它会和前后的道歉、修补、继续协作一起读。;
记录者:这一章给玩家的任务，是辨认“事情评价”和“整体评价”的边界。;

记录者:第三章，重新放回共同叙事。;
记录者:同一个人，前面被否定，后面又重新被放进项目、创作、安排和公共事务里。;
记录者:材料显示，争执并没有让共同叙事立刻中断。;
同学A:也就是说，关系不是在一句话里结束的。;
记录者:对。它会被解释，被转向，被暂时放下，然后继续成为“我们”的一部分。;
记录者:这一章要让玩家看见，伤人的表达和共同目标可以同时存在。;

记录者:第四章，4月9日。;
记录者:旁观者视角里，出现过“什么都能接受”这样的判断。;
记录者:这不是判决，只是旁人对长期承接位置的一种观察。;
旁观者:旁人说的话可靠吗？;
记录者:旁人不是全知视角。它只能说明，关系里的某种姿态已经被外部注意到。;
记录者:4月9日的作用，是把私聊里的疲惫，放到旁观者也能感到不对劲的位置。;

记录者:第五章，4月19日。;
记录者:材料显示，这一天不是一场短争执，而是从凌晨拖到下午的长时间节点。;
记录者:聊天中出现自责、害怕、反复道歉、报复与原谅讨论，也出现持续安抚。;
同学A:这一章要证明谁对谁错吗？;
记录者:不。它只让玩家看见强烈情绪怎样被一次次带回吃饭、休息、礼物和日常安排。;
记录者:4月19日不是单次崩溃，也不是单次和好，而是一段安抚循环再次出现的过程。;

记录者:第六章，分开之后。;
记录者:材料显示，分开之后，相关话题仍在聊天中出现。;
记录者:5月1日前后仍有强烈情绪，线下活动、见面安排、组织事务和商品消费也继续出现。;
记录者:截图呈现，现实安排没有因为关系变化立刻消失。;
记录者:这让玩家看到，私人情绪和共同事务并不是两条完全分开的线。;
记录者:玩家需要分清，哪些是高压表达，哪些是事务沟通，哪些只是物件留下的后续。;
记录者:我看见截图里的字，也看见截图之外仍然缺席的语气、停顿和表情。;
记录者:所以本章不让玩家替任何人宣判，只让玩家标记事情为什么没有自然停止。;
记录者:到这里，时间线不再追问一句话的重量，而是追问分开之后，生活怎样继续要求人回复。;

旁观者:所以这页不是结论页。;
记录者:对。它是索引。玩家从这里进入章节，再回到材料本身。;
记录者:如果某一段只来自截图，就按截图呈现；如果只来自聊天，就按聊天中出现。;
记录者:如果只是旁观者视角，就把它留在旁观者视角。;
记录者:这就是时间线页的规则。;

:时间线页结束。;
```
